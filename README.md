# Thực hành Thống kê

## Giới thiệu

Repo này lưu trữ các R script được thực hiện trong học phần **Thực hành Thống kê Toán học (MTH10109)**, ngành Khoa học Dữ liệu - Trường Đại học Khoa học Tự nhiên, ĐHQG-HCM.

## Nội dung

| Chương | Nội dung chính |
|---|---|
| Chương 1 | Giới thiệu R & RStudio; Các kiểu, cấu trúc dữ liệu cơ bản |
| Chương 2.1 | Working directory, import dữ liệu (csv, txt) |
| Chương 2.2 | Tidy data, `tidyr` (pivot_longer, pivot_wider) |
| Chương 2.3 | Pipeline (`%>%`), `dplyr` (filter, select, arrange, mutate, group by, ...) |
| Chương 2.4 | Literate programming, cấu trúc file `.Rmd`, YAML header, code chunks |
| Chương 2.5 | Grammar of Graphics, trực quan hóa dữ liệu với `ggplot2` |
| Chương 3.1 | Mô phỏng và trực quan các phân phối xác suất rời rạc/liên tục |
| Chương 3.2 | Định lý giới hạn trung tâm (Central Limit Theorem) |
| Chương 4 | Ước lượng điểm (MoM, MLE) và ước lượng khoảng (confidence interval) |
| Chương 5 | Kiểm định giả thuyết thống kê (t-test, prop-test, ...) |
| Chương 6 | Hồi quy tuyến tính đơn (Simple Linear Regression - OLS) |
